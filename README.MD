## Cómo configurar?

Directorios <br>
api6ug8 -> backfront api restful <br>
web6ug8 -> frontend <br>
bot6ug8 -> Bot <br>

Configurar la localizacion `./web6ug8/anything.papajhons`
```c
{
    "webpath": "http://localhost/web6ug8/",
    "apipath": "http://localhost/api6ug8/",
}
```

Acceso a la base de datos `./api6ug8/anything.papajhons`
```c
{
    "mysql": {
        "driver": "mysql",
        "host": "localhost",
        "port": 3306,
        "user": "admin",
        "pass": "admin",
        "database": "diproavilm"
    }
}
```
